# 🧠 Algorithmic Thinking Lab

> A structured, self-paced DSA & Competitive Programming journey — from foundations to interview mastery.

---

## 📚 What Is This?

This repository is a **comprehensive learning lab** for mastering Data Structures, Algorithms, and Problem-Solving techniques. It's organized progressively so you can build skills from the ground up and track your growth over time.

Whether you're preparing for **coding interviews**, **competitive programming contests**, or simply sharpening your **algorithmic thinking**, this repo has you covered.

---

## 🗂️ Repository Structure

| Folder | Description |
|--------|-------------|
| `00_Foundations/` | Core CS fundamentals — complexity analysis, recursion, bit manipulation, math |
| `01_Data_Structures/` | Arrays, Linked Lists, Trees, Graphs, Heaps, Tries, and Advanced DS |
| `02_Algorithms/` | Searching, Sorting, DP, Greedy, Graph Algorithms, String Algorithms |
| `03_Problem_Solving_Patterns/` | Common patterns — Monotonic Stack, Prefix Sum, Binary Search on Answer |
| `04_Platform_Practice/` | LeetCode, GFG, CodeStudio solutions organized by difficulty |
| `05_System_Level_DS/` | Design problems — LRU Cache, Rate Limiter, Concurrency DS |
| `06_Competitive_Programming/` | Codeforces, AtCoder, ICPC Prep, and reusable templates |
| `07_SDE/` | Blind 75, NeetCode 150, Striver SDE Sheet, Company-wise problems |
| `08_Notes_and_Theory/` | Conceptual notes — Big O, DP, Graph Theory, Trees, Interview Mistakes |

---

## 🚀 Getting Started

1. **Clone the repo:**
   ```bash
   git clone https://github.com/yourusername/algorithmic-thinking-lab.git
   cd algorithmic-thinking-lab
   ```

2. **Start with Foundations** (`00_Foundations/`) if you're new to DSA.

3. **Pick a topic** from `01_Data_Structures/` or `02_Algorithms/` and work through the problems.

4. **Use patterns** from `03_Problem_Solving_Patterns/` to recognize problem types faster.

---

## 📝 How to Use Each Folder

Each topic folder typically contains:
- `README.md` — Concept explanation and key insights
- Solution files (`.cpp`, `.py`, `.java`, etc.) with detailed comments
- `notes.md` — Personal notes, edge cases, gotchas

### File Naming Convention
```
<PlatformTag>_<ProblemNumber>_<ProblemName>.<ext>

Examples:
  LC_001_Two_Sum.cpp
  GFG_Detect_Cycle.py
  CF_1800A_Solution.cpp
```

## 🎯 Learning Roadmap

```
Phase 1: Foundations
  └── Time/Space Complexity → Recursion → Bit Manipulation → Math Basics

Phase 2: Core Data Structures
  └── Arrays → Strings → Linked List → Stack → Queue → Hashing

Phase 3: Trees & Graphs
  └── Binary Tree → BST → Heap → Trie → BFS/DFS → Shortest Path → MST

Phase 4: Algorithm Techniques
  └── Binary Search → Sorting → Sliding Window → Two Pointers → Greedy

Phase 5: Dynamic Programming
  └── 1D DP → 2D DP → DP on Strings → DP on Trees → Bitmask DP

Phase 6: Advanced Topics
  └── Segment Tree → Fenwick Tree → String Algorithms → Network Flow

Phase 7: SDE & Interview Prep
  └── Blind 75 → NeetCode 150 → Striver Sheet → Company-wise Practice
```
